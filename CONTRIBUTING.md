# Contributing to PokePvP Cyber Arena

First off, thanks for taking the time to contribute! 🎉

## How Can I Contribute?

### 🐛 Reporting Bugs

- Check [existing issues](../../issues) first to avoid duplicates.
- Open a new issue using the **Bug Report** template.
- Include steps to reproduce, expected vs. actual behavior, browser/OS, and screenshots if relevant.

### 💡 Suggesting Features

- Open an issue using the **Feature Request** template.
- Explain the problem you're trying to solve, not just the solution.

### 🔧 Submitting Changes

1. **Fork** the repository.
2. **Create a branch** from `main`:
   ```bash
   git checkout -b feat/short-description
   ```
3. **Make your changes.** Since this project is currently a single `index.html`, keep edits scoped and test in-browser before committing.
4. **Commit** using clear, conventional messages:
   ```bash
   git commit -m "feat: add ranked matchmaking timer"
   git commit -m "fix: correct HP bar overflow on mobile"
   ```
5. **Push** your branch and open a **Pull Request** against `main`.
6. Fill out the PR template, link related issues, and describe what you tested.

## Style Guidelines

- Keep HTML/CSS/JS inline and dependency-free unless a change genuinely requires a library.
- Match the existing dark, neon "cyber arena" visual theme.
- Prefer small, focused PRs over large sweeping changes.
- Test on both desktop and mobile viewport widths before submitting.

## Code of Conduct

This project follows a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.

## Questions?

Open a [Discussion](../../discussions) or an issue — we're happy to help.
