## 📌 About

**PokePvP Cyber Arena** is a self-contained, single-file web app for real-time, strategic monster battles. Connect a wallet, pull your team, and fight in ranked matches, random-team modes, or custom lobbies — no build step required.

### ✨ Features

| Feature | Description |
|---|---|
| 🏆 **Ranked Matches** | Competitive PvP with live standings |
| 🎲 **Random Teams** | Jump into a battle with a randomly assembled squad |
| 👥 **Custom Lobbies** | Create or join private matches with friends |
| 🦊 **Wallet Login** | Connect a Solana wallet to bring in your holdings |

## 🚀 Quick Start

### Option 1 — Open directly

Just open `index.html` in any modern browser. No install, no build.

### Option 2 — Clone & serve locally

```bash
git clone https://github.com/<your-username>/pokepvp-cyber-arena.git
cd pokepvp-cyber-arena

# serve with any static server, e.g.
python3 -m http.server 8080
# then visit http://localhost:8080
```

### Option 3 — GitHub Pages

This repo is set up to auto-deploy `index.html` to GitHub Pages via the workflow in [`.github/workflows`](.github/workflows). Enable Pages in your repo settings (Source: GitHub Actions) and every push to `main` will publish automatically.
